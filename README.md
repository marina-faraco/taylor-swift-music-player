# Taylor Swift Music Player (API Project)

Aplicação que consome uma API pública para buscar músicas da Taylor Swift,
exibir detalhes das faixas e permitir navegação. Criado para demonstrar uso de
requisições HTTP e manipulação de dados JSON.

## Tecnologias
- Python
- Requests
- JSON

## Como executar
pip install -r requirements.txt
python src/app.py
